wpCssFramework

wpCssFramework is a WordPress Plugin, which “only” includes Bootstrap (BS) Columns, and also the styles I use for project work. It’s important to note that this frameworks intended use case is for custom themes, i.e. those that are being created, not those that have already been created.

While it's intended use case is as a WordPress Plugin, it can also be used on its own, as can be seen here, wpCssFramework is used to style this demo page.

Columns

Stripped back the BS Columns to those I most commonly use, i.e. .col-4 .col-6 .col-8 .col-12, and also sizes .col-sm-{} .col-md-{} .col-lg-{} .col-xl-{}
