wpCssFramework

A tiny footprint CSS framework, that I use for commercial projects to speed up development work. wpCssFramework was created as a WordPress Plugin, but can as easily be used for HTML / CSS projects, and includes a mish-mash of styles that I most commonly use, mostly my creations, but also a little from bootstrap and foundation.

Only wpCssFramework was used to style this page.

Columns

Stripped back the BS Columns to those I most commonly use, i.e. .col-4 .col-6 .col-8 .col-12, and also sizes .col-sm-{} .col-md-{} .col-lg-{} .col-xl-{}
